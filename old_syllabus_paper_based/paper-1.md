## Practice Paper 1
### Questions

**1. Which of the following protocols is primarily responsible for the secure transmission of data over the internet?**  
A) FTP  
B) HTTP  
C) HTTPS  
D) SMTP  

**2. In the URL `http://www.sliit.lk:8080/index.html`, what does `8080` represent?**  
A) The Protocol  
B) The IP Address  
C) The Directory structure  
D) The Port number  

**3. Which statement accurately describes the difference between an Intranet and an Extranet?**  
A) An Intranet represents the World Wide Web, while an Extranet is a local network.  
B) An Extranet is an Intranet that allows controlled access to authorized external users.  
C) An Intranet uses physical cabling, whereas an Extranet is fully wireless.  
D) There is no difference; they are interchangeable terms for Wi-Fi.

A) To translate human-readable domain names into IP addresses  
B) To protect the server from SQL injections  
C) To host HTML and CSS files for a website  
D) To format hard drives over a network  

**5. Which form submission method is considered better for sending sensitive data because it does not append the data to the URL?**  
A) GET  
B) RETRIEVE  
C) SEND  
D) POST  

**6. Which HTML element is used to create a numbered list?**  
A) `<ul>`  
B) `<list>`  
C) `<nl>`  
D) `<ol>`  

**7. Identify the correct HTML syntax for embedding an image.**  
A) `<image src="pic.jpg" alt="A picture">`  
B) `<img src="pic.jpg" alt="A picture">`  
C) `<img href="pic.jpg" title="A picture">`  
D) `<picture link="pic.jpg">`  

**8. If an HTML `<form>` tag does not explicitly specify a `method` attribute, what is the default behavior?**  
A) It will not submit any data.  
B) It defaults to the POST method.  
C) It defaults to the GET method.  
D) It triggers a JavaScript error.  

**9. Which HTML element defines a navigation section containing links?**  
A) `<links>`  
B) `<nav>`  
C) `<menu>`  
D) `<navigation>`  

**10. What is the essential structural difference between `<div>` and `<span>` elements?**  
A) There is no difference; they behave identically.  
B) `<div>` is used for text, `<span>` is used for images.  
C) `<div>` is an inline element, `<span>` is a block-level element.  
D) `<div>` is a block-level element, `<span>` is an inline element.  

**11. How do you create an email hyperlink in HTML?**  
A) `<a href="mail:user@example.com">`  
B) `<a href="mailto:user@example.com">`

A) External Stylesheet

D) Browser default styles  

**16. How do you make all text inside `<p>` elements bold using CSS?**  
A) `p { font-weight: bold; }`  
B) `p { style: bold; }`

A) The zoom level of an image.  
B) The horizontal alignment of elements.  
C) The stacking order of overlapping elements along the Z-axis.
