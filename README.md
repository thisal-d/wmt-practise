# Web and Mobile Technology – Mid Exam Practice MCQs

## Introduction
Welcome to the Mid-Semester Exam practice repository for the **Web and Mobile Technology** module. This repository contains full-length practice MCQ papers designed to simulate the actual machine-based mid-semester examination.

## Exam Information
- **Date:** 14th March 2026
- **Format:** 25 Multiple-Choice Questions (MCQs) per paper
- **Duration:** 1 Hour per paper
- **Scope:** Lecture 1 to Lecture 5
- **Topics Covered:** Web Concepts, HTML, CSS, and JavaScript

## Topics Covered Details
* **Web Concepts:** Client-Server architecture, URL components, DNS resolution, networking definitions, and common web protocols (HTTP, HTTPS, FTP, SMTP, POP3).
* **HTML:** Structural tags, form methods (GET vs POST), lists, tables, images, hyperlinks, and specific attributes.
* **CSS:** Selectors, inclusion methods (Inline, Internal, External), text & color formatting, Box Model (margin, padding, border), and layout styling.
* **JavaScript:** DOM manipulation (`getElementById`), variable declarations, logical and mathematical operators, loops (`for`, `while`), arrays, basic events, and type coercion.

## Practice Papers (Based on 2026 Model Paper)

We have created **3 comprehensive practice papers**. 
**Why 3 papers?**
- **Paper 1:** Focuses heavily on direct concept testing across Web, HTML, CSS, and JS. Ideal for establishing core knowledge.
- **Paper 2:** Introduces moderate complexity, introducing more "Identify the Output" and "Spot the Error" type questions mirroring the actual style.
- **Paper 3:** A rigorous, fully balanced simulation with tough edge cases, DOM events, and specific HTML/CSS nesting scenarios seen in the model paper. 
This progression perfectly simulates a student's final preparation cycle.
