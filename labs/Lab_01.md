## Introduction

Website is a collection of web pages which are grouped together and usually connected in various ways. A web page is a document commonly written in Hypertext Markup Language (HTML) that is accessible through the Internet or other network using an Internet browser. A web page is accessed by entering a URL address and may contain text, graphics, and hyperlinks to other web pages and files. HTML stands for Hypertext Markup Language, and it is the most widely used language to write Web Pages. HTML describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. HTML elements are represented by tags.
